## 🤖 AI Agent Guidelines (Prompting Context)

When generating code, copy, or structural components for Integrative, adhere to the following standards:

### ⚡ Design Principles
- **Conversion First:** UI/UX must be lean, optimized for high-speed loading and user-action conversion.
- **High Contrast:** Utilize sharp visual hierarchies and deep contrast for maximum readability.
- **Responsive Layouts:** Mobile-first, fluid designs for all breakpoints.

### 🏗️ Recommended Web Structure
1. **Hero Section:** High-impact value proposition.
2. **Service Bifurcation:** Distinct separation between "Custom Solutions" and "Products/SaaS".
3. **Evidence:** Portfolio showcases and technical use cases.
4. **CTAs:** Direct and frictionless (e.g., "Agendar llamada", "Ver planes").

### 💻 Technical Stack
- **Frameworks:** React (Next.js preferred).
- **Styling:** Utility-first CSS (Tailwind CSS) with clean modular components.
- **Code Quality:** Deliver production-ready, modular code. Exclude unnecessary comments. Focus on dry (Don't Repeat Yourself) principles.

---

> [!TIP]
> **Focus on Agility.** Every line of code and every pixel should serve a functional purpose. Minimalist aesthetics do not mean simple; they mean refined.
